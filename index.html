Dicionário para iniciantes no Java script

Esse é um dicionário de Java Script onde vou passar alguns código Java script e explicar para o que cada um serve para usar junto com os códigos HTML e CSS.

1.window(alert, prompt, confirm)
  • Quando usar? Quando quisermos criar uma janela ao entrar no site, de alerta, pergunta ou digitação
  • Para que serve? Serve para interagir com o usuário, exibindo mensagens, solicitando informações ou confirmando ações.
  • Exemplo Prático:
html
<div class="demo">
<!-- Botões que chamam funções JavaScript para mostrar alertas, confirmação e prompt -->
<button onclick="mostrarAlert()"> Mostrar Alerta </button>
<button onclick="mostrarConfirm()"> Mostrar Confirmação </button>
<button onclick="mostrarPrompt()"> Mostrar Pergunta </button>

</div>
<script>
    function mostrarAlert() {
     window.alert("Olá, seja bem-vindo ao meu dicionário de Java script!"); // Exibe um alerta com uma mensagem
    }
    function mostrarConfirm() {
    window.confirm("Quer ter acesso a esse conteúdo?"); // Exibe uma caixa de confirmação com uma pergunta
    }
    function mostrarPrompt() {
    window.prompt("Me fale seu nome e vamos começar!"); // Exibe uma caixa de entrada para o usuário digitar algo
    }
</script>

O que significa isso?

function mostrarAlert() → Cria uma função chamada mostrarAlert.
window.alert("...") → Exibe uma caixa de alerta com a mensagem escolhida.
function mostrarConfirm() → Cria uma função chamada mostrarConfirm.
window.confirm("...") → Exibe uma caixa de confirmação com a opção de escolher OK ou Cancelar.
function mostrarPrompt() → Cria uma função chamada mostrarPrompt.
indow.prompt("...") → Exibe uma caixa de entrada para o usuário digitar algum texto.
  • function → Cria um bloco de código reutilizável com ações.
  • window → Representa a janela do navegador.
  • alert → Mostra uma mensagem simples ao usuário.
  • confirm → Pergunta ao usuário e retorna verdadeiro (OK) ou falso (Cancelar).
  • prompt → Permite que o usuário digite uma resposta.

Resumindo:

Essas funções mostram caixas de diálogo diferentes no navegador: alert para avisos, confirm para perguntas de confirmação e prompt para pedir entradas de texto do usuário.

Quando usar:

Você pode usar esses comandos quando precisar interagir diretamente com o usuário, como exibir mensagens, pedir confirmação antes de executar algo, ou solicitar dados.

2.getElementById()
  • Quando usar? Procura no documento um elemento HTML com base na ID especificada
  • Para o que serve? Permite acessar e manipular o elemento selecionado pela ID dele usando JavaScript.
  • Exemplo prático:

Meu Código Java Script

Quando clicar o botão abaixo, esse texto vai mudar

<h3>Meu Código Java Script </h3>

<!-- Parágrafo com id "teste" que será alterado pelo botão -->

<p id="teste"> Quando clicar o botão abaixo, esse texto vai mudar </p>

<!-- Botão que altera o texto do parágrafo acima -->  

<button type="button" onclick="document.getElementById('teste').innerHTML=('Texto Mudou')"> Clique aqui</button>

Explicação do código:

  • onclick: Serve para indicar que quando der o clique, a ação a seguir vai acontecer.
  • document.getElementById("teste"): Aqui ele indica que ele vai procurar no documento, um elemento cuja a ID é "teste".
  • innerHTML = ("Texto mudou") : o innerHTML ele vai obter o código HTML interno, e definir que quando clicar o botão ele vai ser "Texto Mudou".

Temos outro exemplo também:

<!-- Botão para ligar a luz (troca a imagem) -->

<button onclick="document.getElementById('minhaimagem') .src='imagens/pic_bulbon.gif'"> Liga a luz </button>

<!-- Imagem da lâmpada, começa desligada -->

<img id="minhaimagem" src="imagens/pic_bulboff.gif" style="width:100px">

<!-- Botão para desligar a luz (troca a imagem) -->

<button onclick='document.getElementById("minhaimagem").src="imagens/pic_bulboff.gif"'>Desliga a luz </button>

Explicação do Código:

  • 1º button: O primeiro <button> o atributo getElementById vai buscar o elemento que vai mudar cuja a id é "minhaimagem" e em seguida um .src com uma imagem com a lâmpada ligada
  • img: Uma imagem que define como vai ficar sem nenhuma alteração, que é a lâmpada desligada com a id "minhaimagem"
  • 2º button: O segundo <button> o atributo getElementById vai buscar o elemento que vai mudar cuja a id é "minhaimagem" e em seguida um .src com uma imagem com a lâmpada desligada

O que significa isso?

Imagem (img) → Mostra a lâmpada apagada no início (src="imagens/pic_bulboff.gif") e tem o id="minhaimagem".
Botão "Liga a luz" → Quando clicado, usa getElementById("minhaimagem") e troca o src para a imagem da lâmpada acesa (pic_bulbon.gif).
Botão "Desliga a luz" → Faz o mesmo processo, mas troca o src para a lâmpada apagada (pic_bulboff.gif).
  • getElementById("minhaimagem") → busca a imagem
  • .src = "imagens/pic_bulbon.gif" → troca a imagem para lâmpada acesa
  • .src = "imagens/pic_bulboff.gif" → troca a imagem para lâmpada apagada

Resumindo:

Esse código controla uma imagem (a lâmpada), trocando o arquivo exibido (src) de acordo com o botão clicado.

Quando usar:

Quando você quiser alterar dinamicamente uma imagem em sua página através de eventos, como clique em botões.

3.getElementsByTagName()
  • Quando usar? Localiza todos os elementos de uma página HTML que possuem uma determinada tag e retorna uma lista deles para que possam ser manipulados com JavaScript.
  • Para o que serve? Localizar elementos pelo nome da tag na página e permitir alterar seu conteúdo ou estilo com JavaScript.
  • Exemplo Prático:

    • Quando eu clicar no botão todos os parágrafos vão ficar vermelhos e mudar o texto

    Oi

    Jossuã

    Andrade

      html
<div id="minhaDiv">
<h3>Quando eu clicar no botão todos os parágrafos vão ficar vermelhos e mudar o texto</h3>

<!-- Três parágrafos que serão modificados pelo botão -->
<p>Oi</p>
<p>Jossuã</p>
<p>Andrade</p>

<!-- Botão que chama a função para modificar os parágrafos -->
<button onclick="mudarParagrafos()"> Clique aqui </button>
</div> 
      <script>
// Função que altera todos os parágrafos dentro da div "minhaDiv"//
  function mudarParagrafos() {
    let div = document.getElementById("minhaDiv");

    // Seleciona todos os parágrafos dentro da div
    let paragrafos = div.getElementsByTagName("p");

    for (let i = 0; i < paragrafos.length; i++) {
      paragrafos[i].style.color = "red";
      paragrafos[i].innerHTML = "Parágrafo modificado!";
    }
  }
</script>

    Explicação do código

    • fuction mudarParagrafos() : Cria uma função que o nome é Paragrafos
    • let div = document.getElementById("minhaDiv") : Cria a variável div que pega todo o conteúdo da div com id "minhaDiv"
    • let paragrafos = div.getElementsByTagName("p") : Cria uma variavel let com o nome paragrafos e pega todos os parágrafos < p > que estão dentro da div
    • for (let i = 0; i < paragrafos.length; i++) : Cria um loop que começa com i=0, e enquanto i for menor que o tamanho da variavel paragrafos, ele vai executar o código dentro do loop, e depois de cada execução, ele incrementa i em 1 (i++)
    • paragrafos[i].style.color = "red" : Muda a cor do texto do parágrafo para vermelho
    • paragrafos[i].innerHTML = "Parágrafo modificado!" Troca o conteúdo do parágrafo pelo texto "Parágrafo modificado!"

    O que significa isso?

    Função Criada → Existe uma função chamada mudarParagrafos().
    Variável div → Guarda o conteúdo da div que tem o id="minhaDiv".
    Variável paragrafos→ Pega todos os elementos < p > que estão dentro dessa div.
    Loop for → Percorre todos os parágrafos.
    • i = 0 → começa no primeiro parágrafo
    • i < paragrafos.length → vai até o último parágrafo
    • i++ → aumenta 1 a cada volta

    Resumindo:

    Esse código pega todos os < p > que estão dentro da div minhaDiv, passa por eles um por um com o for e altera o estilo (cor vermelha) e o texto (mostrando “Parágrafo modificado!”).

    Quando usar:

    Esta função seria chamada através de um evento (como o clique de um botão) para transformar visualmente todos os parágrafos de uma seção específica da página.

    4.getElementsByClassName()
    • Quando usar? Procura no documento elementos com base na Class especificada
    • Para o que serve? Permite acessar e manipular os elementos com CLASS selecionado usando JavaScript.
    • Exemplo Prático:
    Olá
    Olá
    Olá
    html
<div class="mudarclass">

<!-- Três h5 que serão modificados pelo botão -->
<h5>Olá</h5>
<h5>Olá</h5>
<h5>Olá</h5>

<!-- Botão que chama a função para modificar a classe -->
<button onclick="mudarClass()"> Clique aqui</button>
</div>

    <script>
// Função que altera o fundo dos elementos com a classe "mudarclass" //
function mudarClass() {

let elementos = document.getElementsByClassName("mudarclass");

for (let i = 0; i < elementos.length; i++) {

elementos[i].style.backgroundColor = "gray";

}
}
</script>

    Explicação do código:

    • function mudarClass() → Cria uma função chamada mudarClass que será executada quando chamada (pelo botão, nesse caso).
    • document.getElementsByClassName("mudarclass") → Seleciona todos os elementos que possuem a classe mudarclass.
    • for (let i = 0; i < elementos.length; i++) → Percorre todos os elementos selecionados, um por um.
    • elementos[i].style.backgroundColor = "gray" → Altera o fundo de cada elemento para cinza.

    O que significa:

      function mudarClass() → Cria a função chamada mudarClass, que só será executada quando for chamada (neste caso, pelo botão). Usamos a função para organizar o código e só executar quando quisermos.
      let elementos = document.getElementsByClassName("mudarclass") → Cria uma variável chamada elementos que guarda todos os itens que têm a classe mudarclass. Como pode haver vários, o JavaScript retorna uma lista (HTMLCollection). Por isso não usamos getElementById, já que o ID só pega um único elemento.
    • for (let i = 0; i < elementos.length; i++) → Cria um loop para percorrer todos os itens que estão dentro da lista elementos. - let i = 0 → Começa no primeiro item da lista. - i < elementos.length → Continua até o último item da lista. - i++ → Faz o contador ir aumentando de 1 em 1, passando por todos os itens. O loop é necessário porque getElementsByClassName retorna vários elementos, então precisamos percorrer todos.
    • elementos[i].style.backgroundColor = "gray" → Pega cada elemento da lista (um por vez, com [i]) e muda o estilo dele. No caso, altera o backgroundColor para cinza. O [i] é importante porque indica qual dos elementos queremos modificar naquele momento do loop.

    Resumindo:

    Ao clicar no botão, o JavaScript pega todos os elementos com a classe mudarclass, percorre a lista e muda o fundo de cada um para cinza. Usamos getElementsByClassName porque há vários elementos e o for para aplicar a alteração em cada item.

    Quando usar:

      Quando você precisa aplicar a mesma mudança em vários elementos (cor, texto, classe, atributo etc.).

      Para ações em lote: destacar vários cards, marcar itens selecionados, esconder/mostrar um grupo.

      Quando os itens compartilham a mesma classe. Se for só um elemento, use getElementById; se quiser seletores CSS mais flexíveis, querySelectorAll(".classe") também funciona.

    5.querySelector()
    • Função: é usada para selecionar o primeiro elemento de um documento HTML que corresponda a um seletor CSS específico
    • Para que serve: Usado para encontrar o primeiro elemento HTML que combina com um seletor CSS (como .classe, #id, ou tag) e permitir alterações no conteúdo, estilo ou eventos desse elemento.
    • Exemplo prático:

    Query Selector

    html
<div class="querySelector">

<!-- Três h3 que serão modificados pelo botão -->
<h3>Query Selector</h3>
<h3>Query Selector</h3>
<h3>Query Selector</h3>

<!-- Botão que chama a função para modificar o segundo h3 -->
<button onclick="mudarQuery()"> Clica ai</button>
</div>
    <script>
//Função que altera o tamanho da fonte do segundo h3 dentro da div "querySelector"//
function mudarQuery() {

  let query = document.querySelector(".querySelector h3:nth-child(2)"); { //Abre aspas para colocar os elementos que serão estilizados//

  query.style.fontSize = "30px"; //Altera o tamanho da fonte do segundo h3 para 30px //
  query.style.color = "red"; //Altera a cor do segundo h3 para vermelho //
}
} //Fecha aspas para fechar a função e a estilização do elemento//
</script>

    Explicação do código:

    • function mudarQuery() → Cria uma função chamada mudarQuery que será executada quando chamada (pelo botão, nesse caso).
    • let query = document.querySelector(".querySelector h3:nth-child(3)") → Seleciona especificamente o terceiro elemento h3 dentro do elemento com classe "querySelector" e armazena na variável query.
    • query.style.fontSize = "30px" → Altera o tamanho da fonte do elemento selecionado para 30 pixels.
    • query.style.color = "red" → Altera a cor do texto do elemento selecionado para vermelho.

    O que significa:

      function mudarQuery() → Cria a função chamada mudarQuery, que só será executada quando for chamada (neste caso, pelo botão). Usamos a função para organizar o código e só executar quando quisermos.

      let query = document.querySelector("querySelector h3:nth-child(3)") → Cria uma variável chamada query que guarda um elemento específico. O querySelector usa seletores CSS para encontrar elementos. Neste caso: - querySelector → Busca elementos com a classe "querySelector" - h3 → Dentro dessa classe, procura elementos h3 - :nth-child(3) → Especificamente o terceiro filho h3 Diferente do getElementById que pega por ID, o querySelector é mais flexível e usa seletores CSS.
    • query.style.fontSize = "30px" → Pega o elemento armazenado na variável query e altera a propriedade CSS fontSize para 30 pixels. O .style permite modificar estilos CSS diretamente via JavaScript.
    • query.style.color = "red" → Pega o mesmo elemento e altera a propriedade CSS color para vermelho. Como já temos o elemento na variável query, podemos aplicar múltiplas mudanças de estilo.

    Resumindo:

    Ao clicar no botão, o JavaScript usa um seletor CSS para encontrar especificamente o terceiro elemento h3 dentro da div com classe "querySelector", depois altera o tamanho da fonte para 30px e a cor para vermelho. Usamos querySelector com seletores CSS avançados para selecionar elementos específicos de forma mais precisa que getElementById.

    Quando usar:

      Quando você precisa aplicar a mudança específica em um único elemento usando seletores CSS complexos como nth-child, classes específicas, etc.

      Para seleções precisas: pegar o segundo item de uma lista, o terceiro parágrafo de uma div, elementos em posições específicas.

      Quando você quer flexibilidade de seletores CSS. Se for só um ID, use getElementById; se quiser vários elementos, use querySelectorAll; para um elemento específico com seletor complexo, use querySelector.

    6.querySelectorAll()
    • Função:É usada para selecionar todos os elementos em um documento HTML que correspondem a um ou mais seletores CSS especificados
    • Para que serve: Serve para selecionar todos os elementos de uma página que correspondam a um ou mais seletores CSS especificados.
    • Exemplo prático:

    Mensagem 1

    Mensagem 2

    Mensagem 3

     html 
<!-- Três parágrafos com a mesma classe para serem modificados pelo botão -->

<p class="mensagem">Mensagem 1</p>
<p class="mensagem">Mensagem 2</p>
<p class="mensagem">Mensagem 3</p>

<!-- Botão para acionar a função -->
<button onclick="alterarMensagens()">Alterar mensagens</button>
    <script>
//Função que altera todos os parágrafos com a classe "mensagem"//
function alterarMensagens() {
  const mensagens = document.querySelectorAll(".mensagem");

  mensagens.forEach((mensagem, index) => {
    mensagem.textContent = `Mensagem alterada ${index + 1}`;
    mensagem.style.color = "blue";
  });
}
</script>

    O que significa:

      function alterarMensagens() → Cria a função chamada alterarMensagens, que só será executada quando for chamada (neste caso, pelo botão). Usamos a função para organizar o código e só executar quando quisermos.

      const mensagens = document.querySelectorAll(".mensagem") → Cria uma variável chamada mensagens que guarda todos os elementos que têm a classe "mensagem". O querySelectorAll retorna uma lista (NodeList) com todos os elementos encontrados. Diferente do querySelector que pega apenas o primeiro, este pega todos os elementos com essa classe.
    • mensagens.forEach(function(mensagem, index)) → Usa o método forEach para percorrer cada elemento da lista mensagens. Para cada elemento: - mensagem → É o elemento atual sendo processado - index → É a posição do elemento na lista (0, 1, 2, etc.) O forEach é uma alternativa moderna ao loop for tradicional.
    • mensagem.textContent = "Mensagem alterada " + (index + 1) → Para cada elemento, altera o conteúdo de texto. O textContent substitui apenas o texto (sem HTML). A expressão (index + 1) soma 1 ao índice para numerar de 1, 2, 3... ao invés de 0, 1, 2...
    • mensagem.style.color = "blue" → Para cada elemento processado, altera a propriedade CSS color para azul. Como estamos dentro do forEach, isso é aplicado a todos os elementos da lista.

    Resumindo:

    Ao clicar no botão, o JavaScript seleciona todos os elementos com classe "mensagem", percorre cada um usando forEach, altera o texto para "Mensagem alterada" seguido de um número sequencial, e muda a cor para azul. Usamos querySelectorAll para pegar múltiplos elementos e forEach para processar cada um de forma moderna e limpa.

    Quando usar:

      Quando você precisa aplicar a mesma mudança em vários elementos que compartilham uma classe comum.

      Para processamento em lote com numeração: alterar múltiplos cards, mensagens, itens de lista com conteúdo sequencial.

      Quando você quer código mais moderno e legível. O forEach é mais limpo que loops for tradicionais e o querySelectorAll oferece flexibilidade de seletores CSS.

    7.innerHTML
    • Quando usar? Para alterar ou obter o conteúdo HTML de um elemento.
    • Para que serve: Serve para ler ou alterar o conteúdo HTML dentro de um elemento, permitindo que você mude texto, tags e até imagens dinamicamente.
    • Exemplo Prático:

    Conteúdo original

    html
<!-- Div que terá seu conteúdo alterado pelo botão -->
<div id="conteudo1">
<p>Conteúdo original</p>
<button onclick="alterarConteudo()">Alterar conteúdo</button>
</div>
    <script>
//Função que altera o conteúdo da div "conteudo1"//
  function alterarConteudo() {
    let conteudo = document.getElementById("conteudo1");

    conteudo.innerHTML = "
    >Novo conteúdo adicionado!
    "; // Altera o conteúdo HTML interno da div
  }
</script>

    Explicação do código:

    • function alterarConteudo() → Cria uma função chamada alterarConteudo que será executada quando chamada (pelo botão, nesse caso).
    • let conteudo = document.getElementById("conteudo") → Seleciona o elemento que possui o ID "conteudo" e armazena na variável conteudo.
    • conteudo.innerHTML = "<h2>Novo conteúdo adicionado!</h2>" → Altera todo o conteúdo HTML interno do elemento selecionado.

    O que significa:

      function alterarConteudo() → Cria a função chamada alterarConteudo, que só será executada quando for chamada (neste caso, pelo botão). Usamos a função para organizar o código e só executar quando quisermos.

      let conteudo = document.getElementById("conteudo") → Cria uma variável chamada conteudo que guarda o elemento HTML que possui o ID "conteudo". O getElementById busca especificamente um elemento pelo seu ID único na página.
    • conteudo.innerHTML = "<h2>Novo conteúdo adicionado!</h2>" → Substitui completamente o conteúdo HTML interno do elemento selecionado. O innerHTML permite inserir código HTML (tags, elementos, formatação), diferente do textContent que insere apenas texto simples. Neste caso, está inserindo um elemento <h2> com o texto "Novo conteúdo adicionado!".

    Resumindo:

    Ao clicar no botão, o JavaScript localiza o elemento com ID "conteudo" e substitui todo seu conteúdo HTML interno por um novo elemento <h2>. Usamos getElementById para selecionar um elemento específico e innerHTML para inserir conteúdo HTML formatado.

    Quando usar:

      Quando você precisa substituir completamente o conteúdo de um elemento específico com novo HTML.

      Para atualizações dinâmicas de conteúdo: trocar seções inteiras, carregar novos componentes, atualizar áreas específicas da página.

      Quando você quer inserir HTML formatado. Use innerHTML para HTML com tags; textContent para texto simples; getElementById para elementos únicos por ID.

    8.style
