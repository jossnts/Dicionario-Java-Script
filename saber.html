  • Index

    • let é uma palavra-chave usada para declarar variáveis em JavaScript. As variáveis declaradas com let têm escopo de bloco, o que significa que elas só são acessíveis dentro do bloco onde foram definidas.

    let nome = "João";

      Pensa no for como um contador que anda pelos botões um por um. Ele tem 3 partes:

    • Inicialização: 
      let i = 0
    • começa o contador 
      i no 0.

      • Por quê 0? Porque listas em JS começam no 0: o 1º botão é botoes[0], o 2º é botoes[1], etc.


    • condição: 
       i < botoes.length

      • Pergunta: “ainda existe botão na posição i?”


      botoes.length é a quantidade total de botões. Enquanto i for menor que esse total, o loop continua.


      Isso garante que o último i usado é length - 1 (o último índice válido).


    • incremento:
       i++
      • depois de cada volta, faz i = i + 1 (vai para o próximo botão).
